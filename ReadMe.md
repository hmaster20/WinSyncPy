# WinSyncPy

WinSyncPy это аналог FreeFileSync,
работающий существенно быстрее и не имеющий проблем синхронизации большого объема мелких файлов на SD карт.

WinSyncPy предназначен для синхронизации каталогов в Windows с поддержкой специфичных для NTFS функций, таких как списки контроля доступа (ACL), альтернативные потоки данных (ADS) и длинные пути.

## Возможности

* **Синхронизация каталогов:** Поддержка режимов `update` (обновление) и `mirror` (зеркалирование).
* **Сохранение атрибутов NTFS:**
  * Копирование ACL (списков управления доступом).
  * Копирование альтернативных потоков данных (ADS).
* **Поддержка длинных путей:** Использование префикса `\\?\` для обхода ограничения в 260 символов.
* **Многопоточность:** Параллельное копирование файлов для повышения производительности.
* **Отказоустойчивое копирование:** Использование временных файлов (`.ws_tmp`) для обеспечения целостности целевого файла в случае ошибки.
* **Повторные попытки:** Обработка временных ошибок ввода-вывода с настраиваемой экспоненциальной задержкой.
* **Логирование:** Подробное логирование в консоль и (опционально) в файл.
* **Исключения:** Возможность исключать файлы и папки по маске.
* **Пробный запуск:** Симуляция операций без их фактического выполнения.
* **Проверка целостности:** Опциональная проверка скопированных файлов с помощью хеша SHA256.
